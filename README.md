# Sito personale - Vincenzo Falbo

Sito statico professionale pronto per GitHub Pages.

## File principali

- `index.html` — homepage completa
- `privacy.html` — privacy policy base
- `cookie.html` — cookie policy base
- `assets/css/style.css` — stile grafico
- `assets/js/script.js` — menu mobile e animazioni
- `assets/img/` — immagini, favicon e ritratto

## Come pubblicarlo su GitHub Pages

1. Crea una repository su GitHub.
2. Carica tutti i file contenuti in questa cartella.
3. Vai su **Settings > Pages**.
4. In **Build and deployment**, scegli:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salva e attendi la pubblicazione.

## Cose da controllare prima della pubblicazione definitiva

- Verificare se è autorizzato l’uso della dicitura “presso Intesa Sanpaolo”.
- Non usare loghi ufficiali della banca senza autorizzazione.
- Far verificare Privacy Policy e Cookie Policy se vengono aggiunti form, analytics, newsletter o mappe incorporate.
- Aggiornare eventuali dati fiscali o recapiti se cambiano.
